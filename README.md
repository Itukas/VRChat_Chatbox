极简的vrc聊天工具，实现VR模式下英文以外的输入

## 🛠️ 环境要求

* Python 3.6 或更高版本
* VRChat (需要开启 OSC 功能)

## 📦 安装步骤

1.  **下载代码**
    将 `main.py` 和 `requirements.txt` 放入同一个文件夹中。
2.  **安装依赖**
    ```bash
    pip install -r requirements.txt
    ```

## 🚀 使用方法

### 1. 开启 VRChat OSC 功能
* 打开 **圆盘菜单->设置->OSC**。
* 确保OSC为开启状态。

### 2. 运行程序
在终端中运行以下命令：

```bash
python main.py
